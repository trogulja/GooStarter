# GooStarter
New project starter based on google sheets. Will read sheets, create new folder & update sheets with new item.

## Required .env
GCLOUD_PROJECT - project ID
GOOGLE_APPLICATION_CREDENTIALS - credentials-file.json
SHEET_SCOPES
SHEET_ID
SHEET_RANGE
